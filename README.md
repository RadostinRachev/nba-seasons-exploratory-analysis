# 🏀 NBA Seasons Exploratory Analysis (Excel)

## 📖 Overview

This project analyzes NBA player statistics across multiple seasons using Microsoft Excel. The goal was to transform raw player data into meaningful insights by answering business-oriented analytical questions through Pivot Tables, formulas, charts, and statistical analysis.

The project demonstrates practical Excel skills commonly used in Data Analyst roles, including data cleaning, aggregation, ranking, and performance evaluation.

---

## 📂 Dataset

The dataset contains NBA player statistics from multiple seasons, including:

- Player information
- Team
- Height & Weight
- Points
- Rebounds
- Assists
- True Shooting Percentage (TS%)
- Usage Percentage (USG%)
- Net Rating
- Offensive & Defensive Rebound Percentage
- Draft information
- Seasons

---

# 🎯 Project Objectives

The analysis answers four main business questions:

### 1. Season Leaders

- Identify scoring leaders
- Identify rebounding leaders
- Identify assist leaders
- Compare their efficiency using TS%, Usage%, and Net Rating

---

### 2. Era Comparison

Compare players from:

- 1990s
- 2000s
- 2010s
- 2020s

Analyze:

- Height
- Weight
- Scoring
- Rebounding
- Assists
- Shooting efficiency

---

### 3. Team & Player Analysis

Determine:

- Which teams consistently produce top performers
- Which player profiles perform best
- Trends in offensive production

---

### 4. Custom MVP Model

Develop a custom MVP Index using normalized player statistics.

The MVP Score combines:

- 40% Scoring
- 15% Rebounding
- 15% Assists
- 30% Shooting Efficiency (TS%)

The results are compared with the official NBA MVP selections.

# 📈 Key Findings

- Elite scorers generally maintain high shooting efficiency despite increased offensive usage.
- Modern NBA players demonstrate higher scoring efficiency compared to previous eras.
- Several teams consistently develop or acquire high-performing players.
- The custom MVP model closely aligns with official NBA MVP selections in many seasons while providing an objective statistical evaluation.
